Detailed Description

Internal typedef struct that is a part of the KS_SEQ_CONTROL typedef struct. Should not be used directly

KS_SEQ_HANDLE is used internally by the KS_SEQLENGTH() macro to define the end of the sequence module in @pg and by ks_scan_playsequence() on TGT during scanning to switch sequence module by calling ks_scan_switch_to_sequence()->boffset()
